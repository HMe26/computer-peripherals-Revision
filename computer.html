âœ… Essay Questions

  1. Q1: Explain the difference between buffering and spooling.
    Answer:

    Buffering is a technique where a small part of RAM is used to temporarily store data when transferring between a peripheral and the CPU. This helps synchronize the speed difference between the fast CPU and slow devices.

    Spooling, on the other hand, sends the data to a secondary storage like a hard disk before sending it to the device (e.g., printer). It allows background printing and frees up CPU to perform other tasks.

  2. Q2: Describe the main differences between CRT and LCD monitors.
    Answer:

    CRT (Cathode Ray Tube) monitors are bulky and use electron beams to light phosphor pixels on the screen, which emits radiation and consumes more power.

    LCD (Liquid Crystal Display) monitors are slim, energy-efficient, and use liquid crystals between polarizing filters to control light and display images. They are safer and more modern than CRTs.

  3. Q3: What are the key components of an IoT system?
    Answer:

    IoT systems are built using:

    • Embedded Systems: Low-power controllers for managing devices.
    • Sensors: For data collection (temperature, motion, etc.).
    • Cloud Computing: For storing and processing large data.
    • Control Units: Microprocessors that run logic.
    • Networking: Wireless or wired internet connectivity to communicate.
