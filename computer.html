🧠 Lecture 1: Computer Peripherals

Computer Peripherals - Quick Overview

Computer peripherals are external devices connected to a computer. They are divided into two categories based on: Function (Input/Output) and Intended use (Memory/Storage). All peripherals are slower than the CPU, so an I/O module is needed as an interface. Types include Human-readable (Screen, Keyboard), Machine-readable (Hard disks), and Communication devices (Modem, NIC).

🔹 English Explanation:

Computer peripherals are external devices connected to a computer.

They are divided into two categories based on:

  • Function: Input/Output (e.g., mouse, printer).
  • Intended use: Memory or storage (e.g., hard disk, floppy disk).

All peripherals are slower than the CPU, which creates a problem when exchanging data. That’s why we need an I/O module to act as an interface between CPU/memory and these devices.

External devices may be:

  • Human-readable: Screen, keyboard.
  • Machine-readable: Hard disks.
  • Communication devices: Modem, NIC (Network Interface Card).

🔹 شرح بالعربي:

الملحقات الطرفية هي أجهزة خارجية متصلة بالحاسوب.

تنقسم إلى قسمين:

  • حسب الوظيفة: إدخال/إخراج (مثل الماوس، الطابعة).
  • حسب الغرض: تخزين أو ذاكرة (مثل الأقراص الصلبة).

كل الأجهزة الطرفية أبطأ من المعالج والذاكرة، لذا نحتاج إلى وحدة I/O (الدخل/الخرج) للربط بينها وبين وحدة المعالجة المركزية.

الأجهزة الخارجية قد تكون:

  • مرئية للبشر: الشاشة، لوحة المفاتيح.
  • مقروءة للآلة: القرص الصلب.
  • أجهزة اتصال: مثل المودم، وكارت الشبكة.

🧠 Lecture 2: Peripheral Devices

Peripheral Devices - Quick Overview

Device Drivers: Programs that allow OS to communicate with hardware. Plug and Play Devices: Auto-detectable by Windows. Input Devices: Keyboards, Pointing Devices (mouse, touchpad), Scanners (OCR, barcode), Voice Input (mic + sound card), Handwriting Input. Output Devices: Monitors (CRT, LCD), Printers (laser, inkjet, dot matrix, plotter), Voice Output (speakers, sound card). Storage Devices: Floppy Disk (1.44MB, magnetic), Hard Disk (High capacity, magnetic), Optical Disk (CD/DVD, laser), Tape (Sequential access, backup), USB/Flash cards.

🔹 English Explanation:

Device Drivers are software programs that help the OS communicate with hardware. Some are built-in (e.g., keyboard driver), while others need manual installation.

Plug and Play (PnP) devices can be automatically detected and configured by Windows.

Input Devices:

  • Keyboard – USB, Wireless.
  • Pointing Devices – Mouse, Touchpad, Light pen.
  • Scanning Devices – Image scanner, Bar code reader, OCR.
  • Voice Input – Microphone + sound card + voice recognition software.
  • Handwriting Input – Needs handwriting recognition software.

Output Devices:

  • Monitor – CRT (bulky, high radiation) and LCD (slim, no radiation).
  • Printers:
    • Laser: High quality, fast, expensive color printing.
    • Inkjet: Colorful, cheaper printer but expensive ink.
    • Dot Matrix: Low quality but cheap, can print multi-copies.
    • Plotter: Draws maps, posters (used in CAD).
  • Sound Output – Speakers and sound card.

Storage Devices:

  • Floppy Disk – 1.44 MB, slow, magnetic.
  • Hard Disk – 100+ GB, high-speed magnetic storage.
  • Optical Disk – CD/DVD, uses laser, 700MB – 17GB.
  • Tape – Sequential access, cheap, used for backup.
  • Other – USB, Zip disks, Flash memory cards.

🔹 شرح بالعربي:

تعريف الدرايفر: هو برنامج يربط بين نظام التشغيل والجهاز (مثل الطابعة، الماوس).

PnP: الأجهزة التي يمكن لويندوز التعرف عليها تلقائيًا بدون تدخل المستخدم.

أجهزة الإدخال:

  • لوحة المفاتيح بأنواعها.
  • أدوات التأشير: ماوس، تراك بول، شاشة لمس.
  • الماسحات: سكانر، OCR، باركود.
  • إدخال الصوت: ميكروفون + كارت صوت + برنامج تعرف صوتي.
  • إدخال باليد: برنامج خاص لتحويل الكتابة اليدوية لنص.

أجهزة الإخراج:

  • الشاشة: CRT (ضارة وكبيرة)، LCD (نحيفة وآمنة).
  • الطابعات:
    • ليزر: عالية الجودة وغالية.
    • إنكجت: ملونة، طابعة رخيصة لكن الحبر غالي.
    • دوت ماتريكس: ضعيفة لكنها تطبع نسخ متعددة.
    • بلوتر: لرسم الخرائط والتصميمات الهندسية.
  • إخراج صوتي: سماعات.

أجهزة التخزين:

  • مرنة (فلوبي).
  • صلبة (هارد).
  • بصرية (CD/DVD).
  • شرائط (للباكاب).
  • أخرى: USB, كروت ذاكرة، Zip.

🧠 Lecture 3–4: Interfaces and Buffering

Interfaces & Buffering - Quick Overview

Interface connects CPU to peripherals and manages: Buffering (temporary data storage), Serial - Parallel conversion, Analog - Digital conversion (ADC/DAC), Voltage & Protocol conversion, Status signal handling. Buffer: RAM, small, fast. Spooler: HDD, big, slower but can work in background.

🔹 English Explanation:

Interface connects CPU with peripherals, solving differences in:

  • Data format.
  • Speed.
  • Voltage.
  • Protocol.

Interface Functions:

  • Buffering: Temporary RAM to store data to match CPU and device speeds.
  • Data Conversion:
    • Serial ↔ Parallel.
    • Analog ↔ Digital (ADC/DAC).
  • Voltage Conversion.
  • Protocol Conversion.
  • Status Signal Handling.

Buffer vs Spooler:

  • Buffer: RAM-based, fast, limited size.
  • Spooler: Uses hard disk, big size, slower.

🔹 شرح بالعربي:

الواجهة (Interface) تربط المعالج بالأجهزة الطرفية وتُعالج الفروقات في السرعة، الشكل، والفولت.

وظائف الواجهة:

  • Buffering: ذاكرة مؤقتة لتخزين البيانات وتنسيق السرعات.
  • تحويل البيانات:
    • تسلسلي ↔ متوازي.
    • تماثلي ↔ رقمي (ADC/DAC).
  • تحويل الفولت.
  • تحويل البروتوكول.
  • التعامل مع إشارات الحالة (مثلاً: الطابعة جاهزة أم لا).

الفرق بين البفر والسبولر:

  • البفر سريع ويستخدم الرام.
  • السبولر أبطأ لكنه يستخدم الهارد وله سعة كبيرة.

🧠 Lecture 5–6: Display Systems (الشاشات)

Display - Quick Overview

Pixel: Smallest display unit. Resolution: Number of pixels (e.g., 1024x768). Display Size: Diagonal (e.g., 15"). Color Depth: Bits per pixel (e.g., 8 bits = 256 colors). Display Types: CRT (Bulky, uses electron guns), LCD (Thin, no radiation). Concepts: Dot Pitch, Aspect Ratio, Video RAM, Interlacing.

🔹 English Explanation:

Basic Display Concepts:

  • Pixel: The smallest unit in an image.
  • Resolution: Number of pixels on screen (e.g., 1024 × 768).
  • Display Size: Diagonal length (e.g., 15”).
  • Color Depth: How many bits per pixel. E.g., 8 bits = 256 colors.

Types of Displays:

  • CRT (Cathode Ray Tube):
    • Big, bulky, uses electron guns.
    • RGB (Red, Green, Blue) phosphor dots.
    • Causes radiation.
  • LCD (Liquid Crystal Display):
    • Thin, no radiation.
    • Uses light polarization and electric current to show images.

Other Display Concepts:

  • Dot Pitch: Distance between pixels; smaller means sharper image.
  • Aspect Ratio: Width to height ratio (e.g., 4:3, 16:9).
  • Video RAM: Stores pixel data (depends on resolution & color depth).
  • Interlacing: Drawing half of the lines each pass to reduce flickering.

🔹 شرح بالعربي:

مفاهيم أساسية:

  • البكسل: أصغر وحدة في الصورة.
  • الدقة: عدد البكسلات على الشاشة (مثلاً: 768×1024).
  • حجم الشاشة: يقاس قطريًا (مثلاً: 15 إنش).
  • عمق اللون: عدد البتات لكل بكسل، 8 بت = 256 لون.

أنواع الشاشات:

  • CRT:
    • كبيرة وضخمة.
    • تستخدم مدافع إلكترونية لعرض RGB.
    • تصدر إشعاعات.
  • LCD:
    • رفيعة وآمنة.
    • تعتمد على تغيير اتجاه الضوء.

مفاهيم إضافية:

  • Dot Pitch: كلما قل كان أوضح.
  • Aspect Ratio: نسبة العرض إلى الارتفاع.
  • Video RAM: ذاكرة لتخزين بيانات الصورة.
  • Interlacing: عرض سطر وترك آخر لتقليل الضغط على الشاشة.

🧠 Lecture 7: Scanner

Scanner - Quick Overview

Scanner digitizes images. Types: Flatbed, Sheet-fed, Handheld.

🔹 English Explanation:

A scanner digitizes physical images into computer format.

Types:

  • Flatbed – Document lies on a glass.
  • Sheet-fed – Sheets are fed into the scanner.
  • Handheld – Moved manually over the page.

Resolution:

  • Measured in DPI (dots per inch), typically 300–600 dpi.

🔹 شرح بالعربي:

الـ Scanner يحوّل الصور الورقية إلى صور رقمية داخل الكمبيوتر.

الأنواع:

  • Flatbed: يوضع المستند على زجاج.
  • Sheet-fed: الأوراق تدخل تلقائيًا.
  • Handheld: يُمرر يدويًا على الورقة.

الدقة:

  • تقاس بـ DPI (نقطة لكل إنش)، عادة 300–600.

🧠 Lecture 8: Printers

Printers - Quick Overview

Types: Impact (dot matrix), Non-Impact (laser, inkjet, thermal). Dot Matrix: Uses pins, prints multi-part forms.

🔹 English Explanation:

Types of Printers:

  • Impact:
    • Dot Matrix (low quality, carbon copies).
  • Non-Impact:
    • Laser: High resolution, fast.
    • Inkjet: Cheaper, colorful, slower.
    • Thermal: For photos.

Dot Matrix:

  • Uses pins to hit ribbon and paper.
  • Speed: CPS (Characters per Second).
  • MTBF: Mean Time Between Failures.

🔹 شرح بالعربي:

أنواع الطابعات:

  • Impact (ضرب):
    • مثل Dot Matrix: جودة ضعيفة لكنها تطبع عدة نسخ.
  • Non-Impact (بدون ضرب):
    • ليزر: جودة عالية.
    • إنكجت: أرخص لكنها أبطأ.
    • حرارية: تستخدم للصور.

Dot Matrix:

  • تعتمد على دبابيس تضرب الشريط والحبر يطبع على الورق.
  • السرعة تقاس بـ CPS.
  • العمر يقاس بـ MTBF.

🧠 Lecture 9: Internet of Things (IoT)

IoT (Internet of Things) - Quick Overview

IoT = Connecting devices to the Internet. Components: Sensors, Embedded systems, Cloud, Control unit, Networking. Applications: Smart homes, health, cities.

🔹 English Explanation:

IoT = Internet of Things

It means connecting physical devices (like fridges, cars, lights) to the Internet.

Components:

  • Embedded Systems – Low-power microcontrollers.
  • Sensors – Collect data (temperature, motion, etc.).
  • Cloud Computing – Stores and processes the data.
  • Control Units – Microprocessors that run logic.
  • Networking – Devices must connect via internet (IP).

Applications:

  • Smart homes, cities, health, security, automation.

🔹 شرح بالعربي:

الـ IoT يعني الإنترنت للأشياء، أي ربط الأجهزة المادية مثل الثلاجات والأبواب بالإنترنت.

المكونات:

  • أنظمة مدمجة – شرائح قليلة الاستهلاك.
  • حساسات – تقيس بيانات مثل الحرارة والحركة.
  • السحابة – لتخزين ومعالجة البيانات.
  • وحدة تحكم – تعالج بيانات الحساسات.
  • الاتصال – لابد من الاتصال بالإنترنت.

التطبيقات:

  • منازل ذكية، مدن ذكية، مستشفيات، أمن، أوتوميشن.

🧠 Lecture 10: Cloud Computing

Cloud Computing - Quick Overview

Cloud = Using internet servers for storage and processing. Types: IaaS (Infrastructure), PaaS (Platform), SaaS (Software). Features: On-demand, scalable, shared resources.

🔹 English Explanation:

Cloud computing = Using online servers to store, process and access data.

Types:

  • IaaS – Infrastructure (e.g., virtual machines).
  • PaaS – Platform (e.g., hosting web apps).
  • SaaS – Software (e.g., Gmail, Google Docs).

Features:

  • On-demand access.
  • Shared resources.
  • Scalability and flexibility.

🔹 شرح بالعربي:

الحوسبة السحابية تعني استخدام الإنترنت للوصول إلى سيرفرات تقوم بالتخزين والمعالجة.

أنواع الخدمة:

  • IaaS: بنية تحتية (سيرفرات افتراضية).
  • PaaS: منصة لتطوير تطبيقات.
  • SaaS: برامج جاهزة مثل Gmail.

الميزات:

  • الوصول عند الحاجة.
  • موارد مشتركة.
  • مرونة وتوسّع.

  1. A buffer stores data permanently for future use.
    False
    Explanation: Buffer stores data temporarily to match device and CPU speeds.
  2. USB is a type of parallel interface.
    False
    Explanation: USB is a serial interface.
  3. Flash memory devices contain no moving parts.
    True
  4. A modem converts digital signals into analog and vice versa.
    True
  5. The resolution of a display is measured in Megabytes.
    False
    Explanation: It is measured in pixels (e.g., 1024×768).

✅ Essay Questions

  1. Q1: Explain the difference between buffering and spooling.
    Answer:

    Buffering is a technique where a small part of RAM is used to temporarily store data when transferring between a peripheral and the CPU. This helps synchronize the speed difference between the fast CPU and slow devices.

    Spooling, on the other hand, sends the data to a secondary storage like a hard disk before sending it to the device (e.g., printer). It allows background printing and frees up CPU to perform other tasks.

  2. Q2: Describe the main differences between CRT and LCD monitors.
    Answer:

    CRT (Cathode Ray Tube) monitors are bulky and use electron beams to light phosphor pixels on the screen, which emits radiation and consumes more power.

    LCD (Liquid Crystal Display) monitors are slim, energy-efficient, and use liquid crystals between polarizing filters to control light and display images. They are safer and more modern than CRTs.

  3. Q3: What are the key components of an IoT system?
    Answer:

    IoT systems are built using:

    • Embedded Systems: Low-power controllers for managing devices.
    • Sensors: For data collection (temperature, motion, etc.).
    • Cloud Computing: For storing and processing large data.
    • Control Units: Microprocessors that run logic.
    • Networking: Wireless or wired internet connectivity to communicate.
